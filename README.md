🛒 CRUD de Produtos em Python

Este projeto consiste em um sistema simples de CRUD (Create, Read, Update, Delete) desenvolvido em Python, utilizando listas e dicionários para armazenar os dados em memória.

Foi criado como atividade acadêmica com foco em lógica de programação e manipulação de estruturas básicas.

🚀 Funcionalidades

O sistema permite:

📋 Listar produtos
➕ Adicionar produtos
🔍 Buscar produtos pelo nome
✏️ Atualizar produtos existentes
❌ Remover produtos
🧠 Conceitos Utilizados
Estruturas de repetição (while, for)
Estruturas condicionais (if, elif, else)
Listas e dicionários
Funções em Python
Entrada e saída de dados (input, print)
⚠️ Observações
Os dados são armazenados apenas em memória (não utiliza banco de dados).
Ao encerrar o programa, todas as informações são perdidas.
O sistema não possui tratamento de erros avançado (entrada inválida, por exemplo).
